# Nmax

Тестовое задание для FunBox

## Установка и использование

    git clone git@github.com:vladen41k/nmax.git
    cd nmax
    export PATH="./bin:$PATH"
    setup
    cat PATH_TO_YOUR_FILE | bundle exec nmax < N >
